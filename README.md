# Lemon-67 - Digital Storefront

This project is the digital storefront for "Lemon-67," a fictional sports drink. The website is designed to introduce the brand, showcase its products, and encourage customers to join the loyalty rewards program.

## Features

- **Brand Introduction**: A compelling hero section to introduce Lemon-67.
- **Product Showcase**: A dedicated section to display the various drink flavors.
- **Benefits Highlight**: Information about the drink's benefits, like hydration and natural energy.
- **Rewards Program**: A sign-up form for the "Lemon-67 Rewards Program" to capture customer interest and build a community.

## Project Structure

- `index.html`: The main HTML file.
- `css/style.css`: All custom CSS for styling the website.
- `js/script.js`: JavaScript for interactive features.
- `images/`: A directory to store all images (logos, product shots, etc.). Note: This project uses placeholder images.
